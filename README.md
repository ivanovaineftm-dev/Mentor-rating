# Mentor rating

Веб-приложение на Flask для загрузки Excel-файла и расчета рейтинга сотрудников.

## Возможности

- Загрузка Excel-файла через простой веб-интерфейс.
- Автоматическая обработка всех листов и всех строк файла.
- Добавление столбца `Балл` сразу после столбца `M`.
- Скачивание обновленного файла с сохранением структуры книги.

## Формула расчета

```text
(0,4 * B + 0,35 * M + 0,25 * E) * (1 + 0,2 * H)
```

Правила интерпретации:

- `B`: пусто — `1`, числовое значение — `0`.
- `M`: числовое значение — `1`, пусто или другое значение — `0`.
- `E`: `сдан` — `1`, любое другое значение — `0`.
- `H`: `сдан` — `1`, любое другое значение — `0`.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После запуска откройте `http://localhost:5000`.
