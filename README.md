# Mentor rating

Веб-приложение на Flask для загрузки Excel-файла и расчета рейтинга сотрудников.

## Возможности

- Загрузка Excel-файла через простой веб-интерфейс без ограничения по расширению имени файла.
- Автоматическая обработка всех листов и всех строк файла.
- Добавление столбца `Балл` сразу после столбца `M`.
- Скачивание обновленного файла с сохранением структуры книги.

## Формула расчета

```text
0,4 * B_coef + 0,2 * M_coef + 0,2 * E_coef + 0,2 * H_coef
```

Правила интерпретации:

- `B`: пусто — `1`, любое заполненное значение — `0`.
- `M`: любое заполненное значение — `1`, пусто — `0`.
- `E`: `пройден` после trim/lower — `1`, любое другое значение — `0`.
- `H`: `сдан` после trim/lower — `1`, любое другое значение — `0`.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После запуска откройте `http://localhost:5000`.
